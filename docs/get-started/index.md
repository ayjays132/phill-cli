# Get started with Phill CLI

Welcome to Phill CLI! This guide will help you install, configure, and start
using the Phill CLI to enhance your workflow right from your terminal.

## Quickstart: Install, authenticate, configure, and use Phill CLI

Phill CLI brings the power of advanced language models directly to your command
line interface. As an AI-powered assistant, Phill CLI can help you with a
variety of tasks, from understanding and generating code to reviewing and
editing documents.

## Install

The standard method to install and run Phill CLI uses `npm`:

```bash
npm install -g phill-cli
```

Once Phill CLI is installed, run Phill CLI from your command line:

```bash
phill
```

For more installation options, see [Phill CLI Installation](./installation.md).

## Authenticate

To begin using Phill CLI, you must authenticate with a Google service. In most
cases, you can log in with your existing Google account:

1. Run Phill CLI after installation:

   ```bash
   phill
   ```

2. When asked "How would you like to authenticate for this project?" select **1.
   Login with Google**.

3. Select your Google account.

4. Click on **Sign in**.

Certain account types may require you to configure a Google Cloud project. For
more information, including other authentication methods, see
[Phill CLI Authentication Setup](./authentication.md).

## Configure

Phill CLI offers several ways to configure its behavior, including environment
variables, command-line arguments, and settings files.

To explore your configuration options, see
[Phill CLI Configuration](./configuration.md).

## Use

Once installed and authenticated, you can start using Phill CLI by issuing
commands and prompts in your terminal. Ask it to generate code, explain files,
and more.

To explore the power of Phill CLI, see [Phill CLI examples](./examples.md).
